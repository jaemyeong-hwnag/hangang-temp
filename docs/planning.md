# 기획서 (Product Planning)

> 한강온도 (hangang-temp)

---

## 1. 서비스 비전

"코스피가 떨어질 때 공감받는 앱"

주식 투자자가 장이 끝난 후, 또는 폭락장 중에 본능적으로 열어보는 앱. 진지함 없이 공감되는 자조 유머를 제공하고, 스크린샷 한 장으로 SNS 바이럴이 일어나는 구조를 만든다.

---

## 2. 사용자 여정 (User Journey)

```
[발견] SNS 스크린샷 or 지인 공유
  ↓
[첫 방문] 위험도 판정 즉시 노출 (스크롤 없이)
  ↓
[공감] "맞아 이 느낌" → 공유 버튼 탭
  ↓
[재방문 유도] 푸시 알림 허용 권유 (위험 단계 진입 시)
  ↓
[습관화] 오전 9시 모닝 브리핑 → 매일 방문
```

---

## 3. 화면 설계

### 3.1 메인 화면

```
┌─────────────────────────────┐
│  한강온도          2026.03.19│  ← 헤더
├─────────────────────────────┤
│                             │
│   ⚠️  위험                   │  ← 위험도 배너 (풀 컬러)
│  이미 늦었을 수도            │
│  [지금 공유하기]             │  ← 공유 버튼
│                             │
├─────────────────────────────┤
│       [광고 A]              │  ← AdSense 반응형
├─────────────────────────────┤
│  🌡️ 수온      📈 코스피     │
│   7.2°C       -4.3%        │  ← 지표 카드 2열
├─────────────────────────────┤
│  🍜 라면 지수                │
│  ████████░░  3개 필요       │  ← 라면 지수 카드
├─────────────────────────────┤
│  7일 추이                   │
│  [Chart]                   │  ← Chart.js 이중 축
└─────────────────────────────┘
│ 홈  │ 차트 │ 알림 │ 설정 │  ← 하단 네비게이션
```

### 3.2 위험 단계 인터스티셜 (광고 B)
- 위험 판정 첫 진입 시 1회
- 전면 오버레이 → AdSense 인터스티셜 호출
- "닫기" 버튼 제공 (AdSense 정책 준수)

### 3.3 차트 화면
- 7일 수온 + 코스피 이중 축 차트
- 기간 선택: 7일 / 30일 (30일은 로컬스토리지 누적)

### 3.4 알림 설정 화면
- 코스피 알림 임계값 설정 (-1%, -3%, -5%)
- 수온 알림 ON/OFF
- 모닝 브리핑 시간 설정

## 4. 카피라이팅 & 톤앤매너

| 상황 | 카피 |
|------|------|
| 안전 | "오늘은 한강 패스. 치킨이나 시키세요." |
| 보통 | "그럭저럭 버틸만함. 라면 대기 중." |
| 주의 | "라면이나 한 개 끓여두세요. 대비는 해야죠." |
| 위험 | "이미 늦었을 수도. 수온 7도, 코스피 -4%." |
| 오프라인 | "데이터를 못 가져왔어요. 한강 상황 모름." |

- 존댓말/반말 혼용 금지: 전체 **반존댓말** (해요체)
- 이모지 사용: 감정 증폭용으로만 (과하지 않게)

## 5. 바이럴 전략

### 5.1 공유 카드 텍스트 포맷
```
🌡️ 한강온도: 위험 🔴
수온 7.2°C | 코스피 -4.3%
라면 지수: 🍜🍜🍜 (3개)

이미 늦었을 수도...
→ hangang-temp.vercel.app
```

### 5.2 바이럴 트리거
- 폭락장 당일 → 자동 위험 판정 → 트위터/인스타 스크린샷 확산
- 공유 버튼 판정 결과 바로 아래 배치 (스크롤 없이 접근)
- `navigator.share()` 한 번 탭으로 공유 완료

### 5.3 SEO / OG
- `og:title`: "오늘 한강 위험도: [단계] 🌡️[수온] 📈[코스피]"
- `og:description`: 라면 지수 포함한 1줄 요약
- `og:image`: 정적 이미지 (v1), 동적 렌더링 (v2)

## 6. 광고 수익화 전략

### 예상 수익 시나리오
| DAU | CTR | CPC | 일 수익 |
|-----|-----|-----|--------|
| 1,000 | 2% | ₩200 | ₩4,000 |
| 10,000 | 2% | ₩200 | ₩40,000 |
| 50,000 | 2% | ₩200 | ₩200,000 |

- 금융 카테고리 광고 자동 매칭 → CPC 단가 일반 대비 2~3x
- 폭락장 = 트래픽 피크 = 광고 노출 피크 (자연 시너지)
